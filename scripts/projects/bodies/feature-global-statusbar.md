背景
- 顶部全局状态条提供运行态“一眼观全局”。

范围
- 展示字段：当前 runId、用时、成功率、缓存命中、活跃日志源计数；
- 与 Run Center/Flow 的选中与链路状态联动；
- 支持点击跳转详情（run/日志源）。

验收
- [ ] 状态条在 `StudioPage` 持续渲染并 1s 以内更新统计；
- [ ] 成功率/缓存命中随事件流实时更新；
- [ ] 点击 runId 跳转 Run 详情；点击日志源打开日志面板。

依赖
- Dexie runs/logs 聚合查询；
- 事件流统计（成功/失败/缓存命中）。

测试与度量
- 组件测试：字段渲染与点击跳转；
- 性能：更新不阻塞主线程（<16ms/帧）。
